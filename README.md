# 📰 News Scraper Project (Eitaa MVP)

یک پروژه‌ی آموزشی و پروژه‌محور برای یادگیری پایتون با تمرکز بر:
- معماری تمیز
- اسکریپ اخبار
- پردازش و انتشار محتوا
- آماده‌سازی خروجی برای پیام‌رسان ایتا (Eitaa)

این پروژه به‌صورت **مرحله‌به‌مرحله و Commit-based** توسعه داده می‌شود
و برای استفاده در دوره‌های آموزشی طراحی شده است.

---

## 🎯 اهداف پروژه
- آموزش پایتون به‌صورت پروژه‌محور
- آشنایی با معماری ماژولار
- جمع‌آوری اخبار از منابع مختلف
- پردازش و استانداردسازی داده
- انتشار خروجی در ایتا (MVP)
- آماده‌سازی برای وب و توسعه‌های بعدی

---

## 🧱 معماری کلی پروژه

```text
news_scraper/
│
├── collector/              # جمع‌آوری اخبار از منابع
│   └── news_collector.py
│
├── sources/                # منابع خبری
│   ├── base.py             # قرارداد منبع خبری
│   └── example_source.py   # منبع تستی
│
├── models/                 # مدل‌های داده
│   └── news.py             # مدل استاندارد خبر (dataclass)
│
├── processor/              # پردازش خبر (تمیزسازی، خلاصه‌سازی و...)
│   └── news_processor.py
│
├── publisher/              # انتشار خروجی
│   └── eitaa_client.py     # کلاینت ارسال به ایتا (MVP)
│
├── storage/                # ذخیره‌سازی (آماده برای DB)
│   └── repository.py
│
├── web/                    # خروجی وب (اختیاری)
│   ├── generator.py
│   └── templates/
│       └── index.html
│
├── config/
│   └── settings.py         # تنظیمات پروژه
│
├── data/                   # داده‌های موقت / تست
│
├── main.py                 # نقطه شروع برنامه
